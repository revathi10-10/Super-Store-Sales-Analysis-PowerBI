# Super-Store-Sales-Analysis-PowerBI

# 📊 Super Store Sales Data Analysis – Power BI

## 📌 Project Overview
This Power BI project analyzes Super Store sales data to provide business insights into sales performance, profitability, customer purchasing behavior, and regional trends. The interactive dashboard helps stakeholders make data-driven decisions.

---

## 🚀 Features
- 📈 Total Sales KPI
- 💰 Total Profit KPI
- 📦 Average Orders KPI
- 🛍️ Sales by Category & Sub-Category
- 💳 Sales by Payment Mode
- 🚚 Sales by Ship Mode
- 🗺️ Sales by State (Map)
- 📅 Sales & Profit Trend Analysis
- 🌍 Region-wise Sales Analysis

---

## 🛠️ Tools Used
- Power BI Desktop
- DAX
- Power Query
- Microsoft Excel (Dataset)

---

## 📊 Dashboard Insights
- Total Sales: **1.57M**
- Total Profit: **175.29K**
- Average Orders: **4.11**
- Technology is the highest-selling category.
- COD is the most used payment method.
- Standard Class is the most preferred shipping mode.
- California generates the highest sales.
